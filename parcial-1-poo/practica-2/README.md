# Práctica 2: Herencia y Reutilización de Código

## Explicación de Herencia
Se aplicó herencia permitiendo que la clase `Admin` herede atributos y métodos de `Usuario`.Esto evita duplicar código y facilita el mantenimiento.

## Diferencias entre Usuario y Admin
**Usuario:** Es la clase base con información general (nombre y correo).
**Admin:** Es una clase especializada que añade la funcionalidad de `getRol()`

## Ejecución
1. Activar Apache en XAMPP.
2. Acceder a `localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-2/index.php`.
