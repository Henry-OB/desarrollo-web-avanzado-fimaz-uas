# Práctica 1: Creación de Clases y Encapsulamiento

## Objetivo
Aplique los fundamentos de la POO en PHP, implementando atributos privados, constructor y métodos de acceso.

## Descripción de la clase
La clase `Usuario` gestiona la información básica de un usuario (nombre y correo) utilizando encapsulamiento para proteger la integridad de los datos.

## Instrucciones de ejecución
1. Clonar el repositorio.
2. Mover la carpeta al directorio de un servidor local (XAMPP/Apache).
3. Acceder a C:\xampp\htdocs\desarrollo-web-avanzado-fimaz-uas\parcial-1-poo\practica-1` desde el navegador.
